# offline-service-requests
# Офлайн-приложение для заявок (PWA)

Веб-приложение для выездных специалистов: создание заявок, работа офлайн, синхронизация с сервером при появлении интернета.

## Технологии

- Frontend: HTML, CSS, JS, PWA, Service Worker;
- Backend: Node.js, Express, файловое хранилище (orders.json)
